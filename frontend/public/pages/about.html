Project description

BrewLine is a full Smart Coffee Capsule Manufacturing & Monitoring System built for a university Industrial Informatics module. It simulates a five‑stage production line for single‑serve coffee capsules and demonstrates modern industrial software practices: animated process visualisation, a SCADA‑style HMI, an inspection / defect classifier, a CMMS for machine health, and an analytics layer that ships metrics to InfluxDB and renders them in Grafana.

The entire site is deployed as a static bundle on GitHub Pages. The Python simulator is included in /simulation/ so the same data flow can be reproduced locally against a real InfluxDB / Grafana stack.

Modules delivered

  • Landing page with live SCADA snapshot
  • Animated five‑stage production simulation
  • SCADA HMI with PVs, gauges, alarm bus
  • Quality control + defect classifier (5 classes)
  • CMMS — health cards, schedule, repair log
  • Analytics — InfluxDB / Grafana integration
  • Documentation & architecture overview
